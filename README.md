# A-topology-and-optimisation-software-

 A topology and optimisation software based on SIPM METHOD
 
 The code may be used in courses in structural optimization where students may be assigned to do extensions such as multiple load-cases, alternative mesh-independency schemes, passive areas, etc. Another possibility is to use the programto developstudentsâ€™ intuition foroptimal design.
